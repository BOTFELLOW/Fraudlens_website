Developer-Friendly API

Integrate FraudLens's predictive power with our simple and robust REST API.

Endpoints

  • Health Check
  • Single Prediction
  • Batch Prediction

Base URL

https://api.fraudlens.com/v1

Health Check

Verify that the API service is up and running.

GET /health

Example Response (200 OK)

{ "status": "ok" }

Single Transaction Prediction

Submit a single transaction's data to receive a real-time fraud prediction.

POST /predict

Request Body

{
    "transactionId": "txn_12345abc",
    "amount": 1000,
    "timestamp": "2025-07-04T23:00:00Z",
    "payer": { "vpa": "user@bank", "ifsc": "BANK0001" },
    "beneficiary": { "vpa": "receiver@otherbank", "ifsc": "OBNK0002" },
    "device": { "ip": "203.0.113.1", "userCount": 1 },
    "location": { "lat": 19.0760, "lon": 72.8777 }
}

Example Response (200 OK)

{
    "transactionId": "txn_12345abc",
    "isFraud": false,
    "riskScore": 0.05,
    "riskLevel": "low",
    "reasonCodes": ["IP_MATCH_LOCATION"]
}

Batch Transaction Prediction

Submit a list of transactions for asynchronous processing.

POST /predict/batch
